Media & Form Submission Demo

Two FREE approaches to collect form data

Approach 1: Google Form
Best For:
  • Quick setup, no coding needed
  • Automatic data collection in Google Sheets
  • Built-in spam protection
  • Free and reliable

Click the button below to open a Google Form (you need to create one and replace the URL)

Open Google Form
Setup Instructions:
  1. Go to forms.google.com
  2. Create your form with required fields
  3. Click "Send" → Copy link
  4. Replace URL in this button
Approach 2: Google Sheets
Best For:
  • Custom form on your website
  • Data goes directly to Google Sheets
  • Free and unlimited submissions
  • Full control over form design
Setup Instructions:
  1. Create a Google Sheet
  2. Go to Extensions → Apps Script
  3. Paste the code (see below)
  4. Deploy as Web App
  5. Copy URL and update script

Google Apps Script Code

Copy this code and paste it in your Google Sheets Apps Script editor:

// Google Apps Script Code - Paste in Extensions > Apps Script

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  var data = JSON.parse(e.postData.contents);
  
  // Add timestamp
  var timestamp = new Date();
  
  // Append row to sheet
  sheet.appendRow([
    timestamp,
    data.name,
    data.email,
    data.phone,
    data.message
  ]);
  
  // Return success response
  return ContentService
    .createTextOutput(JSON.stringify({
      'result': 'success',
      'message': 'Data saved successfully'
    }))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService
    .createTextOutput("The script is working!")
    .setMimeType(ContentService.MimeType.TEXT);
}

// Set up headers (run this once)
function setupSheet() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.getRange(1, 1, 1, 5).setValues([
    ['Timestamp', 'Name', 'Email', 'Phone', 'Message']
  ]);
}
Important: After pasting, click Deploy → New Deployment → Web App → Set "Who has access" to "Anyone" → Deploy → Copy the URL
